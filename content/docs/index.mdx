# Welcome

Welcome to the Guest-to-Guest platform documentation! This is the main landing page for our SaaS event management solution.

## What is Guest-to-Guest?

Guest-to-Guest is an intelligent SaaS platform for event and invitation management, designed for individuals, businesses, and professional event planners.

### Key Features

- **AI-Powered Invitations**: Intelligent writing assistant, automatic design generation, and smart suggestions
- **Real-Time Guest Management**: Track RSVPs, manage check-ins, and analyze participation
- **Multi-Platform Support**: Web PWA and mobile applications (React Native/Flutter)
- **Advanced Analytics**: Real-time dashboards and predictive insights

1. Visit the homepage at `/`

## Technical Stack

- **Frontend**: Next.js + TailwindCSS
- **Mobile**: React Native
- **Backend**: Laravel
- **Database**: Mysql
- **AI**: OpenAI API, TensorFlow
